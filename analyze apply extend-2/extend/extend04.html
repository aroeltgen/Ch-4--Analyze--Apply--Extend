HWB summary: Overall, HWB aims to simplify the process of working with colors by providing a model that closely aligns with the way humans perceive and understand color properties.

CMYK summary: CMYK stands for Cyan, Magenta, Yellow, and Key (Black), and it is a color model primarily used in color printing. This model is also referred to as the process color model because it describes the printing process itself. CMYK is a subtractive color model, which means it works by partially or entirely subtracting the light that would otherwise be reflected.
